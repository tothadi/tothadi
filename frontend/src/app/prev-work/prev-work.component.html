RDF Weightbridge

Displaying data emitted by a weightbridge

The purpose of this small program is to show real-time weight data of incoming and outgoing garbage trucks and to store the weights with timestamp. The weightbridge communicates through RS232 with an ATC-1000 RS232/LAN converter. I use the TCP stream emitted by the ATC-1000 to fetch weights and save them to a MongoDB Cloud database. Hosted on my company's physical server with MS Windows Server 2016.
